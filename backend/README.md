# AI Pandit - BTR Engine (Backend)

Self-contained BTR engine with Swiss Ephemeris.

## Environment Variables
Make sure to add the following to **Settings > Variables and Secrets**:
- `TURSO_DATABASE_URL`
- `TURSO_AUTH_TOKEN`
- `CLERK_SECRET_KEY`
- `CLERK_PUBLISHABLE_KEY`
- `NEXT_PUBLIC_CLERK_PUBLISHABLE_KEY`
- `NODE_ENV=production`
